# Job Scheduling projekts

Optimizācijas tīmekļa serviss darbu plānošanai, izmantojot Spring Boot un Timefold.

## Prasības

Java 17+
Maven

## Atkarību lejupielāde

mvn clean install

## Palaišana

mvn spring-boot:run

Vai palaist klasē:
SchedulerApplication.java

## Pēc palaišanas:

http://localhost:8080/piemeri.html
